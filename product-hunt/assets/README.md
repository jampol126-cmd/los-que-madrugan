## Cómo usar

### Opción 1: Screenshot (Más fácil)
1. Abrir cada archivo HTML en Chrome
2. Hacer zoom al 100%
3. Screenshot con herramienta del sistema
4. Recortar a 1270x760px exactos
5. Guardar como PNG

### Opción 2: Convertir con Node.js
```bash
npm install -g puppeteer
node convert-to-png.js
```

### Opción 3: Herramientas online
- https://htmlcsstoimage.com
- https://convertio.co/es/html-png/
